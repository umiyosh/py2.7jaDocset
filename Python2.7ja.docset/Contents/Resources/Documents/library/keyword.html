31.6. keyword — Python キーワードチェック¶

このモジュールでは、 Python プログラムで文字列がキーワードか否かをチェックする機能を提供します。

keyword.iskeyword(s)¶

s が Python のキーワードであれば真を返します。

keyword.kwlist¶

インタープリタで定義している全てのキーワードのシーケンス。特定の __future__ 宣言がなければ有効ではないキーワードでもこのリストには含まれます。
